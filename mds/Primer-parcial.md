# Examen Primer Parcial

#### ¿Qué es y para qué sirve Visual Studio Code? 
Visual Code nos sirve para modificar carpetas y tambien para crear en estas los archivos y modificarlos

#### ¿Qué es y para qué sirve la Terminal de Comandos? 
La terminal de comandos nos sirve para hacer ajustes dentro de un archivo, carpeta o de nuestra computadora para crear ya sea un nuevo archivo, crear carpetas, modicar archivos o carpetas y para tambien subir cosas en un repositorio

#### ¿Qué es y para qué sirve Markdown?
Es una herramienta en la cual nos permite escribir pero tambien darle formarmato al texto como poner imagenes o abrir un archivo de la carpeta todo con un codigo para ejecutar

#### ¿Qué es y para qué sirve Git?
Pues es para tener otra terminal de comandos en la cual la podemos subir en un repositorio de internet o que se quede en la computadora pero modificando las carpetas

#### ¿Qué es y para qué sirve GitHub? 
Pues es un repositorio de internet en la cual nosotros podemos enviar archivos y carpetas, nos sirve para que lo podemos compatir y que este mismo lo vuelvas a subir con los cambios

#### ¿Para qué sirven los siguientes comandos: pwd, whoami, touch, mkdir, cp, mv, ls, clear, cd y rm? 
Pues en el caso de pwd sirve para ver en tu terminal de comandos en que carpetas te ubicas, el mkdir sirve para crear archivos y carpetas, cp es para copiar y pegar archivos, mv es para mover archivos, ls es una lista de todos los archivos que tienes en la carpeta donde estas ubicado, cd es para donde te quieres ir de una carpeta y rm es para remover un archivo.

#### ¿Qué significan los siguiente caracteres en la terminal de Comandos: ./, ../, /, y ~? 
./ es para retroceder un lugar, el ../ es para retroceder dos lugares y / es para buscar una carpeta.

#### ¿Cómo se inicializa un repositorio en Git? 
Primero le vas añadir a agregar repositorio con la cual te va pedir el nombre de ese repositorio y si quieres una licencia, si va ser publico o privado y que lenaguaje vas a ocupar en ese repositorio

#### ¿Cómo creas un repositorio en GitHub? 
Primero es guardando toda tu carpeta con archivos con add y despues tienes que agregar un comentario despues de ese comentario vas a github y vas agregar un repositorio desoues de eso githbu te va mandar un link en la cual va ser tu repositorio de internet y ya despues haces el git push con el link que te envio y ya estara tu repositorio.

#### ¿Cómo vinculas un repositorio local de Git con uno remoto en GitHub? 
Pues va ser con un link que nos va enviar github y despues nostros vamos usar git con la carpeta que queremos enviar y nos va pedir que vinculemos las cuentas y despues de a verlos vinculado ya te va reconocer github y lo va subir al repositorio 

#### ¿Cuál es el flujo básico de trabajo en Git y GitHub?
Pues es con la terminal de comandos ya que ahi puedes subir todo en github y vinculando la cuenta entoces serviria github como un almacen de git

#### ¿Para qué sirve el archivo .gitignore?
Para ignorar archivos que tu coloques ahi si va ser .exe, .md, .log, .jpg, etc, o tambien con nombres del archivo pero que los archivos tengan el mismo nombre.

#### ¿Cuál es el propósito de una rama? 
El proposito de una rama es para que todo el trabajo tenga una parte y tambien para que no se vinvulen algunos archivos

#### ¿Qué es una fusión? 
Cuando tu carpeta tiene cambios en la cual los suben y alguien mas tiene otros archivos y pues tendra que actulizar con el pull y aparecen 4 opciones para combinar tu carpeta nueva con la que tenias

#### Explica los diferentes tipos de fusión que existen.
Existen 4 tipo de fusiones las cuales una lo que hace es fusionar osea no va reemplazar nada y todo lo va subir con los demas archivos, sobreespuesta que esta va poner primero mis archivos que hice, reemplazar donde la carpeta va reemplazar todo lo que hicimos y solo va agregar lo nuevo y por ultimo la que quieres quedarte con tus archivos y no reemplazar ninguno

#### ¿Cómo puedes ver el historial de tu repositorio? 
Para ver el historial del repositorio tienes que es que te tienes que meter al repositorio y ahi te abrir una terminal donde tu puedes ver todo lo que hicieron los comentarios y los tags

#### ¿Cuál es el propósito de una etiqueta? 
Pues su proposito es crear actulizaciones en las cuales tu vas a nombrar y en cada una vas a poder ver el nombre de la actulizacion dependiendo de la version
